About the Project

Understanding the machine learning behind cardiovascular disease prediction.

What is Machine Learning?

Machine Learning (ML) is a branch of artificial intelligence in which algorithms learn patterns directly from data instead of being explicitly programmed. In supervised classification, a model is trained on labelled examples and then predicts the label for new, unseen inputs. This project uses ML to learn the relationship between clinical parameters and the presence of heart disease.

Heart Disease Prediction

Cardiovascular disease is a leading cause of death worldwide. Early risk assessment can support timely intervention. By analysing indicators such as age, blood pressure, cholesterol, chest pain type and ECG results, the trained model estimates the likelihood that a patient has heart disease and returns a probability score.

The Dataset

The dataset follows the well-known UCI heart-disease schema with 13 clinical features and a binary target (0 = healthy, 1 = disease). Features are cleaned, encoded and scaled before training.

Algorithms Used

Five classification algorithms are trained and compared; the best-performing one is saved for predictions.

  • Logistic Regression — linear probabilistic classifier.
  • Decision Tree — rule-based hierarchical splits.
  • Random Forest — ensemble of many decision trees.
  • Support Vector Machine — maximum-margin classifier.
  • K-Nearest Neighbor — instance-based, distance-driven.
